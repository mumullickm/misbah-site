# misbah-site

Privacy policy and support pages for the **Misbah** game, served by GitHub Pages.

- `index.html` — support / about page
- `privacy.html` — privacy policy (the app collects nothing)

## Hosting

Served from GitHub Pages on the `mumullickm/misbah-site` repo. Once Pages is on:

- Support: `https://mumullickm.github.io/misbah-site/`
- Privacy: `https://mumullickm.github.io/misbah-site/privacy.html`

Use the privacy URL in the Google Play Console listing and Data Safety form.

## Important

This repo **must stay public**. GitHub Pages stops serving a private repo on the
free plan, which would 404 the privacy URL and break the store listing.

## Updating

Edit the HTML and push to the default branch. Pages redeploys automatically.
